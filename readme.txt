Instructions for Use:

The Widget and the Custom Post Type
The widget displays up to four posts from the custom post type that it created in a set order, and will also display the featured image for each post. To add a post to the custom post type, go on the WordPress Dashboard and click the 'Design' menu item. Then click the 'Add New' button next to the page name, write up your post and click the publish button when you're done! When adding posts to the custom post type, remember to set a featured image as well. After all this, you should be able to see your post displayed in the widget.

To set this widget so that it appears on your website, go to 'Appearance' -> 'Widgets' and drag the widget titled 'Custom Posts' to your desired widget area. You can also specify the title that you want the widget to have when it appears on your site and then hit the 'Save' button.

The Shortcodes
Related Posts Shortcode
This shortcode creates a related posts section for the post that it is placed in. To use the shortcode on a post, place a single tag [rp_shortcode] in the desired place in your post's content. This shortcode can also be placed in your theme's single post template. Since this shortcode displays related posts based off of post category, make sure to assign categories to your posts when creating them.

Signature Shortcode
This extra shortcode creates a signature that you can place at the bottom of posts and pages. The user can specify the signature's signing off statement and the author name, the signature's background color, font color, and top border color (the font and border color will always be the same). To use this shortcode place a single tag [signature_shortcode] in your content. To specify the signature's signing off statement, assign a value to statement and to specify the author name, assign a value to author in the shortcode. For example, [signature_shortcode statement="Sincerely," author="Sam"] would produce the signature 'Sincerely, Sam' on your page. To specify the signature's background color, font color, and top border color; you can also assign a value to background and colour. For example, [signature_shortcode background="#DCB19F" colour="#000000"].
